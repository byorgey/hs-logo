hs-logo

Logo turtle graphics interpreter

About hs-logo

hs-logo is written by Deepak Jois.

The interpreter is written in the Haskell programming language, and uses diagrams for describing and rendering images. The author is a relative newcomer to Haskell, so any suggestions and contributions are welcome. Head over the to the github repository for hs-logo to download the sources.
